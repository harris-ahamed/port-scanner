# Port Scanner

A Python-based TCP port scanner that identifies open ports on a target system.

## What it does
Scans ports 1-1000 on a target IP address and reports which ports are open.

## How to use
```bash
python3 port_scanner.py
```

## Example output
Checking port 135...
Port 135 is OPEN!
Checking port 445...
Port 445 is OPEN!

## Skills demonstrated
- Python socket programming
- TCP networking concepts
- Port scanning methodology

## Disclaimer
Only use on systems you own or have permission to scan.
